# Configuration

Replica loads configuration from an optional config file and environment variables. Supported config file names are 
`config.json`, `config.yaml`, `config.yml`, and `config.toml`. Set `CONFIG_FILE` to load a specific config file path.

Environment variables override config file values. Empty environment variables are ignored.

Duration values use Go duration syntax such as `15s`, `10m`, or `8h`.

Storage profiles configure credentials and connection settings for cloud storage providers. 
Profiles live under `storage.profiles` and are keyed by profile name. 
Profile names are case-insensitive: names from files and environment variables are normalized to lowercase, 
so `aws` and `AWS` refer to the same profile.

There is no limit on the number of profiles. Each profile supports only the fields documented below, and every field is optional. 
Storage profiles have no built-in defaults. Environment variables for profiles use 
`STORAGE_PROFILES_<PROFILE_NAME>_<FIELD_NAME>`, for example `STORAGE_PROFILES_AWS_ACCESS_KEY_ID` or 
`STORAGE_PROFILES_BACKBLAZE_ENDPOINT`.

| Config option                                       | Environment variable                                | Type            | Default value             | Description                                                                                                                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------|-----------------|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| -                                                   | `CONFIG_FILE`                                       | string          | unset                     | Optional environment-only path to a config file. When unset, Replica searches for `config.json`, `config.yaml`, `config.yml`, then `config.toml` in the current working directory.                              |
| `app.node_id`                                       | `APP_NODE_ID`                                       | string          | `node-1`                  | Node identifier used by this Replica process. Must not be empty.                                                                                                                                                |
| `app.coordinator`                                   | `APP_COORDINATOR`                                   | bool            | `true`                    | Enables coordinator mode for this process. At least one of `app.coordinator` or `app.storage` must be true.                                                                                                     |
| `app.storage`                                       | `APP_STORAGE`                                       | bool            | `true`                    | Enables storage-node mode for this process. At least one of `app.coordinator` or `app.storage` must be true.                                                                                                    |
| `app.coordinator_url`                               | `APP_COORDINATOR_URL`                               | string          | empty string              | Coordinator base URL used by storage nodes. Required when `app.storage` is true and `app.coordinator` is false.                                                                                                 |
| `app.node_address`                                  | `APP_NODE_ADDRESS`                                  | string          | empty string              | Address this storage node reports to the coordinator for node-to-node communication. Required when `app.storage` is true and `app.coordinator` is false.                                                        |
| `app.heartbeat_interval`                            | `APP_HEARTBEAT_INTERVAL`                            | duration string | `10m`                     | Interval between storage-node heartbeat reports. Must be greater than zero when storage mode is enabled.                                                                                                        |
| `app.api_request_timeout`                           | `APP_API_REQUEST_TIMEOUT`                           | duration string | `15s`                     | Timeout for API requests made by storage nodes. Must be greater than zero when storage mode is enabled.                                                                                                         |
| `app.file_transfer_timeout`                         | `APP_FILE_TRANSFER_TIMEOUT`                         | duration string | `30m`                     | Timeout for file transfer operations. Must be greater than zero when storage mode is enabled.                                                                                                                   |
| `sharing.thumbnail_sizes`                           | `SHARING_THUMBNAIL_SIZES`                           | integer array   | `[256, 512, 1024]`        | Allowed thumbnail sizes. Config files use an integer array. Environment values may be a comma-separated list such as `256,512,1024` or a JSON array such as `[256,512,1024]`. Values must be greater than zero. |
| `sharing.thumbnail_default_size`                    | `SHARING_THUMBNAIL_DEFAULT_SIZE`                    | int             | `256`                     | Default thumbnail size used when a request does not specify a thumbnail size.                                                                                                                                   |
| `sharing.thumbnails_generate_for_video`             | `SHARING_THUMBNAILS_GENERATE_FOR_VIDEO`             | bool            | `true`                    | Enables thumbnail generation for video files.                                                                                                                                                                   |
| `sharing.ffmpeg_path`                               | `SHARING_FFMPEG_PATH`                               | string          | `ffmpeg`                  | Path or executable name used to invoke FFmpeg.                                                                                                                                                                  |
| `sharing.video_inline_max_size_mb`                  | `SHARING_VIDEO_INLINE_MAX_SIZE_MB`                  | int             | `25`                      | Maximum video size, in MB, eligible for inline video handling.                                                                                                                                                  |
| `sharing.video_playback_enabled`                    | `SHARING_VIDEO_PLAYBACK_ENABLED`                    | bool            | `true`                    | Enables shared video playback support.                                                                                                                                                                          |
| `sharing.thumbnail_storage`                         | `SHARING_THUMBNAIL_STORAGE`                         | string          | `/tmp/replica_thumbnails` | Directory used to store generated thumbnails.                                                                                                                                                                   |
| `sharing.thumbnail_storage_limit_mb`                | `SHARING_THUMBNAIL_STORAGE_LIMIT_MB`                | int             | `500`                     | Thumbnail storage size limit, in MB.                                                                                                                                                                            |
| `auth.jwt_secret`                                   | `AUTH_JWT_SECRET`                                   | string          | `change-me`               | Secret used for JWT signing. Required unless running storage-only mode without coordinator mode.                                                                                                                |
| `auth.node_secret`                                  | `AUTH_NODE_SECRET`                                  | string          | empty string              | Plaintext node secret used by storage nodes when authenticating to the coordinator. Required when `app.storage` is true and `app.coordinator` is false.                                                         |
| `auth.access_token_duration`                        | `AUTH_ACCESS_TOKEN_DURATION`                        | duration string | `30m`                     | Lifetime for access tokens. Must be greater than zero when coordinator validation is active.                                                                                                                    |
| `auth.refresh_token_duration`                       | `AUTH_REFRESH_TOKEN_DURATION`                       | duration string | `8h`                      | Lifetime for refresh tokens. Must be greater than zero when coordinator validation is active.                                                                                                                   |
| `auth.share_api_token_cache_duration`               | `AUTH_SHARE_API_TOKEN_CACHE_DURATION`               | duration string | `5m`                      | Duration for caching share API token validation results.                                                                                                                                                        |
| `http.address`                                      | `HTTP_ADDR`                                         | string          | `:8080`                   | HTTP listen address. Required unless running storage-only mode without coordinator mode.                                                                                                                        |
| `database.driver`                                   | `DB_DRIVER`                                         | string          | `sqlite`                  | Database driver. Supported values are `sqlite` and `postgres`. Required unless running storage-only mode without coordinator mode.                                                                              |
| `database.dsn`                                      | `DB_DSN`                                            | string          | see below                 | Database connection string. Required unless running storage-only mode without coordinator mode.                                                                                                                 |
| `database.auto_migrate`                             | `DB_AUTO_MIGRATE`                                   | bool            | `true`                    | Enables automatic database migrations at startup.                                                                                                                                                               |
| `seed.admin_name`                                   | `SEED_ADMIN_NAME`                                   | string          | `admin`                   | Initial admin username used by seed logic.                                                                                                                                                                      |
| `seed.admin_password`                               | `SEED_ADMIN_PASSWORD`                               | string          | `change-me`               | Initial admin password used by seed logic.                                                                                                                                                                      |
| `storage.profiles.<profile_name>.access_key_id`     | `STORAGE_PROFILES_<PROFILE_NAME>_ACCESS_KEY_ID`     | string          | none                      | Access key ID for the named storage profile.                                                                                                                                                                    |
| `storage.profiles.<profile_name>.secret_access_key` | `STORAGE_PROFILES_<PROFILE_NAME>_SECRET_ACCESS_KEY` | string          | none                      | Secret access key for the named storage profile.                                                                                                                                                                |
| `storage.profiles.<profile_name>.region`            | `STORAGE_PROFILES_<PROFILE_NAME>_REGION`            | string          | none                      | Region for the named storage profile.                                                                                                                                                                           |
| `storage.profiles.<profile_name>.endpoint`          | `STORAGE_PROFILES_<PROFILE_NAME>_ENDPOINT`          | string          | none                      | Endpoint for the named storage profile.                                                                                                                                                                         |

Default database dsn:  
`replica.db` for `sqlite`  
`host=localhost user=postgres password=postgres dbname=replica port=5432 sslmode=disable` for `postgres`
